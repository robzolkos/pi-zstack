# pi-zstack

My personal Pi stack as a meta package.

This package exists so I can install my preferred Pi setup in one step.

## Included packages

- `pi-skill-model-effort`
- `pi-slopchop`
- `pi-disable-model-skill-invocation`

## Install

```bash
npm install pi-zstack
```

Or globally:

```bash
npm install -g pi-zstack
```

## Purpose

`pi-zstack` is intentionally minimal: it only declares dependencies on my favorite Pi packages.
